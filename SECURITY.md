# Security Policy

## Supported Versions

This project is a research and educational tool and does not have formal version support.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

You can:
- Open an issue (for non-sensitive issues)
- Or contact the author directly via GitHub

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact

---

## Scope

This project is intended for simulation and educational purposes only.

It does not process real user data or connect to external systems.

---

## Security Considerations

- No sensitive data should be used in the application
- The machine learning model is for simulation purposes only
- Results should not be treated as real-world risk assessments

---

## Disclaimer

This project is designed for research, simulation, and educational purposes. 
While it provides useful insights into cyber risk modelling, it should be complemented with real-world data and expert analysis in production environments.
